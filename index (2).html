什么是JavaScript闭包？

点击卡片翻转查看答案

JavaScript · 中等难度

闭包是指有权访问另一个函数作用域中变量的函数，通常是在一个函数内部创建另一个函数。

function outer() {
  const x = 10;
  function inner() {
    console.log(x); // 可以访问x
  }
  return inner;
}
